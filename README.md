<p align="center">

---

## 🚀 Overview

This project consists of a **pixel-perfect recreation of a webpage** based solely on a provided screenshot.

The objective was to translate a static visual reference into a fully functional and responsive web layout, focusing on **precision, structure, and styling accuracy**.

---

## 🖼️ Before vs After

| Reference Design | Implementation |
|----------------|---------------|
| Screenshot provided | Final web page built with code |

> The layout, spacing, typography, and colors were carefully analyzed and reproduced to match the original design as closely as possible.

---

## ✨ Key Features

- 📐 Pixel-perfect layout accuracy  
- 🎨 Pure CSS styling (no frameworks)  
- 📱 Responsive design adaptation  
- 🧱 Clean and semantic HTML structure  

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Layout, positioning, and styling  

---

## 🧠 What I Learned

- How to break down a design into reusable components  
- Advanced CSS layout techniques (Flexbox, positioning, spacing)  
- Improving attention to detail (margins, paddings, font sizes)  
- Translating visual hierarchy into code  

---

## ⚡ Challenges

- Achieving **pixel-perfect spacing and alignment**  
- Matching fonts and typography without exact specs  
- Recreating complex layouts from a static image  
- Ensuring responsiveness without original design guidelines
